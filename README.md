# Derivatives
This is a website where I explain calculus concepts <br>
So basically I make lessons which are explained in depth with a video I recorded, unit tests which test comprehension of the previous lessons, and an extra lesson which explains the concept briefly and doesn't go into much detail <br>
This is the first time I used MathJax, and the first time building a project lesson by lesson, not brute forcing everything :3 <br>
So first, I started making the home page, styling the lesson grid, text, and background along the way. Since I wanted the project to have a consistent style, I used the same CSS file for all pages, and whenever I wanted something different, I added it to the CSS file. Afterwards, I coded the HTML and JS of each lesson one by one, and whenever I was free and able to record, I recorded as many lessons as possible. <br>
The lessons go like this: Limits => unit test => basics of differentiation => unit test => chain rule and its applications => unit test => advanced differentiation topics and integration. <br>
The hardest part was adding the charts; it took like 5h of research until I watched a video of someone using Google Charts. <br>
I used AI mainly to help with debugging JS errors. <br>
Hope you like the website :3
